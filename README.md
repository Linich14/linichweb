## ¿Que es esta pagina/app?

Esta pagina web la creo con el principal objetivo de seguir practicando mis conocimientos de desarrollo web, jugando con tematicas a mi gusto
como es una pagina web de auto presentacion y ayuda, ya que en esta publicare el materia de apoyo que he creado siendo ayudante de ciertas asignaturas en la 
Universidad Catolica de Temuco, todo esto complementando con imagenes IA que he generado con mi propio modelo(LORA) y desde mi propia computadora.

Si bien esta pagina no sera mi portafolio final, servira de complemento para este y ademas servira como referencia en el tiempo que me falta para completar mi periodo universitario.
Espero seguir añadiendo cosas segun vaya aprendiendo o bien pase el tiempo como dejar links y referencias utiles tanto para la gente que ayudo en las asignaturas como para
amigos o conocidos que visiten mi pagina web.
